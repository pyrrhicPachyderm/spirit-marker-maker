Spirit Marker Maker

For the MJ & iakona TTS mod.

How to use

  1. Drag an image of your spirit onto the blank marker image left.

  2. Move/resize the image to fit in the upper-right hole. It will be automatically mirrored in the lower-left hole.

  3. Save the image using the button above.

  4. In TTS, make a copy of an existing spirit marker. Rename the copy with your spirit's name.

  5. Right click the spirit marker and select "Custom". Change the "Diffuse/Image" to the image you saved from here. If TTS asks whether you'd like to update matching custom objects, select "Cancel".
